關於我

你好！我是一位專注於數據工程與商業智慧 (BI) 的專家。在數據爆炸的時代，我的使命是將雜亂無章的原始資料，轉化為具備商業洞察價值的戰略資產。

我擅長建構穩定的資料管道 (Data Pipeline)，並利用深厚的 SQL Server 經驗確保資料的高效存取與安全性。透過 Power BI 與精密的 DAX 建模，我能將複雜的業務指標轉化為直觀的可視化儀表板；同時，我亦能發揮 Python 的開發潛力，進行自動化腳本撰寫與進階數據分析，為企業解決關鍵的營運難題。
